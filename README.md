# Network Security Lab â€“ Spoofing

Repository ini berisi laporan praktikum **Advanced Network Security** yang membahas simulasi dan analisis serangan keamanan jaringan pada lingkungan **LAN**. Praktikum ini menunjukkan bagaimana protokol tidak aman seperti **HTTP** dan **Telnet** dapat diserang menggunakan teknik **ARP Spoofing**, **Session Hijacking**, dan **IP Spoofing**.

---

## Isi Praktikum
- **ARP Spoofing** (Man-in-the-Middle / MITM)
- **Session Hijacking** pada Telnet
- **IP Spoofing**
  - Ping of Death  
  - SYN Flood  
  - Teardrop  
  - Land Attack  

---

## Tools yang Digunakan
- Kali Linux
- Bettercap
- Wireshark
- hping3
- Telnet dan OpenSSH

---

## Tujuan Praktikum
- Memahami cara kerja serangan spoofing di jaringan
- Mengetahui risiko penggunaan protokol tidak terenkripsi
- Menganalisis dampak serangan **MITM** dan **DoS**
- Mempelajari cara pencegahan serangan jaringan

---

## Mitigasi Keamanan
- Menggunakan protokol aman **SSH** dan **HTTPS**
- Static ARP dan Dynamic ARP Inspection (DAI)
- Firewall dengan Ingress dan Egress Filtering
- Unicast Reverse Path Forwarding (uRPF)

---

## Catatan
Repository ini dibuat **hanya untuk keperluan pembelajaran dan tugas kuliah**.
